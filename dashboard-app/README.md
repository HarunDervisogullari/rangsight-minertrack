# Ransight Dashboard

Ransight is a comprehensive admin dashboard built on **Next.js and Tailwind CSS** providing everything needed for data-driven back-end management and analytics.

![Ransight - Dashboard Preview](./banner.png)

With Ransight, you get access to all the necessary dashboard UI components, elements, and pages required for a complete admin panel. Whether you're managing a complex SaaS platform or a simple admin interface, Ransight provides the tools you need.

Ransight utilizes the powerful features of **Next.js 15** including server-side rendering (SSR), static site generation (SSG), and seamless API route integration. Combined with **React 19** and **TypeScript**, Ransight delivers a robust solution for modern web applications.

## Technology Stack

Ransight is built with modern technologies:

- **Next.js 15**
- **React 19**
- **TypeScript**
- **Tailwind CSS**

## Quick Start

### Prerequisites

Make sure you have Node.js LTS version or higher installed on your machine.

### Installation

1. **Clone the repository:**

```bash
git clone https://github.com/ransight/dashboard.git
```

2. **Navigate to the project directory:**

```bash
cd dashboard
```

3. **Install dependencies:**

```bash
npm install
```

4. **Start the development server:**

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

Ransight provides a complete dashboard solution with:

### Dashboard Pages

- **Main Dashboard:** Overview of key metrics and data
- **Analytics Dashboard:** Detailed analytics and reporting
- **Marketing Dashboard:** Marketing-focused metrics and tools
- **CRM Dashboard:** Customer relationship management interface

### Authentication System

- **Sign In:** User authentication with modern design
- **Sign Up:** User registration with form validation
- **Password Recovery:** Secure password reset functionality

### Components

- **Charts & Graphs:** Advanced data visualization
- **Forms:** Multiple form layouts and input types
- **Tables:** Responsive data tables with sorting
- **UI Elements:** Buttons, badges, alerts, modals, and more
- **Pages:** Profile, settings, calendar, and management pages

### Development Features

- **TypeScript Support:** Full type safety and IntelliSense
- **Responsive Design:** Mobile-first approach
- **Dark Mode:** Built-in theme switching
- **Component-Based:** Modular architecture for easy customization
